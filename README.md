# Lattice-based Kinetic Monte Carlo (KMC) simulation framework:
<p>Time-dependent KMC for surface catalysis. Written to benchmark the computational efficiency of each method. Reference systems are a single molecule TPD and TPR C-H activation of methane in the presence of surface oxygen on a PtCu single-atom alloy catalyst</p>

- Implementations of both the Direct method and Next Reaction method
- Reaction system will vary by version but typically aimed at a single-molecule TPD system
- Contains implementations of flat, stepped and single-atom alloy surfaces for both triangular and square lattices
